# Contributing

This repository is currently in planning mode.

When contributing:

- Keep all content in English.
- Keep the main narrative centered on AgentOps as an operating model.
- Treat AgentOps Toolkit as an optional implementation component, not as the workshop topic.
- Do not add lab implementation content until the lab plan is approved.
- Keep GitHub Pages navigation front matter up to date.
- Add observability considerations to every lab where runtime behavior, traces, telemetry, releases, or incidents are relevant.
