# Security

Do not commit secrets, credentials, tenant-specific data, customer data, private endpoint details, or production telemetry exports to this repository.

Before publishing:

- Review all screenshots and traces for sensitive data.
- Replace tenant names, subscription IDs, user identifiers, and resource names with safe examples.
- Confirm that sample data is synthetic or approved for public use.
- Confirm that links to internal SharePoint or private repositories are removed or clearly marked as private references.
